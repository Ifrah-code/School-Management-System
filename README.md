A full-stack School Management System that helps manage students, teachers, classes, subjects, attendance, and more using Spring Boot, Thymeleaf, JavaScript, and MySQL.

Features:-
1] Admin authentication and authorization using Spring Security (login-protected admin panel).
2] Student management: add, edit, delete, and list students with validation and search/filter options.
3] Class and section management, including basic timetable/structure configuration.
4] Attendance tracking and basic reporting for students.
5] Integration with a geolocation API to capture or validate school/user location where required.
6 ]Persistent storage using MySQL with proper relational mappings via JPA/Hibernate.
